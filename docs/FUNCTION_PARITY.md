# MoYuStock 功能对齐清单

| 原功能 | Flutter 状态 | 说明 |
| --- | --- | --- |
| 自选股分组 | 已实现 | 支持新增、重命名、删除、排序、切换当前分组 |
| 自选标的管理 | 已实现 | 支持股票/币种添加、删除、组内排序；卡片默认贴近 `design-proj`，删除通过悬停/长按确认 |
| 本地持久化 | 已实现 | 使用 `shared_preferences` 保存分组、当前分组和主题 |
| 股票搜索 | 已实现 | 东方财富 + Yahoo Finance 合并搜索 |
| 加密货币搜索 | 已实现 | 内置主流币对 + Binance 精确查询 |
| 实时行情轮询 | 已实现 | 前台每秒刷新当前分组行情，页面退后台后暂停 |
| 股票报价 | 已实现 | 东方财富批量报价，包含涨跌幅、开高低收、成交额、估值指标 |
| 加密货币报价 | 已实现 | Binance 24 小时报价 |
| Yahoo 海外标的 | 已实现降级 | 使用移动端 HTTP 会话，不依赖 Chrome 扩展 |
| 分时图 | 已实现 | 使用 `fl_chart` 折线/面积图，包含均价线与昨收参考线 |
| 日/周/月 K 线 | 已实现 | 使用 `fl_chart` 蜡烛图，叠加 MA5/10/20/30/60 均线 |
| K 线交互 | 已实现 | 支持横向拖拽平移、双指缩放、双击重置视窗 |
| 成交量副图 | 已实现 | K 线下方显示轻量成交量柱，颜色随涨跌变化 |
| 图表缓存 | 已实现 | 分时缓存 30 秒，K 线缓存 10 分钟，减少重复切换请求 |
| 详情指标 | 已实现 | 首屏按 `design-proj` 展示 6 项主指标，点击指标卡可查看完整指标 |
| 明暗主题 | 已实现 | Material 3 + `flex_color_scheme`，默认浅色视觉对齐 `design-proj` |
| 导入导出 JSON | 已实现 | 导出走系统分享，导入走文件选择器 |
| 应用可见性轮询控制 | 已实现 | 使用 Flutter 生命周期控制轮询 |
| Android/iOS 平台工程 | 已实现 | 已生成并纳入 git，可继续做真机打包验证 |
| Web 验证 | 已验证 | 已在 `http://127.0.0.1:8088/` 验证列表、详情、分时、日 K，控制台无警告/错误 |

## 当前重点缺口

1. Android SDK 仍需安装 cmdline-tools 并接受 licenses，之后可做 APK/真机完整验证。
2. iOS 打包需要 macOS + Xcode 环境验证。
3. 图表还可以继续补十字光标、成交量 tooltip、盘口/五档等专业行情细节。
4. 继续按 `design-proj` 做细节校准，目标是首屏与详情页版式、颜色、间距、控件形态保持 95% 以上一致。
